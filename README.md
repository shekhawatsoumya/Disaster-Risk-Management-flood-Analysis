# Disaster Risk Management - Flood Analysis

## Project Overview
This project analyzes flood disaster risk using Excel-based data preparation and analysis. It focuses on understanding flood-prone areas, key risk factors, and actionable insights to support disaster preparedness and mitigation.

## Data Source
- Source: Kaggle dataset
- Data includes historical flood incidents, geographic attributes, and related risk variables.

## Data Cleaning
- Tool: Excel / Power Query Editor
- Steps performed:
  1. Load raw dataset from `Raw Data/`.
  2. Remove duplicate rows and irrelevant columns.
  3. Handle missing values by applying appropriate data imputation or removal.
  4. Standardize column names and data formats (dates, numeric values, categories).
  5. Filter records for relevant years and regions.
  6. Export cleaned data to `Clean Data/` for analysis.

## Analysis (Excel)
- Tool: Excel
- Analysis performed:
  - Descriptive statistics (mean, median, count, standard deviation) for flood-related variables.
  - Trend analysis over time to identify increasing flood frequency and severity.
  - Geographic risk comparison across districts/regions.
  - Correlation checks between rainfall, elevation, population density, and flood occurrences.
  - Create charts and pivot tables for visual insights.

## Results and Findings
- Key flood risk drivers identified from exploratory analysis.
- High-risk zones mapped using conditional formatting and pivot charts.
- Recommendations for disaster risk reduction include stronger early warning, improved drainage, and targeted community resilience.

## Project Structure
- `Raw Data/` - Original downloaded dataset files.
- `Clean Data/` - Cleaned dataset ready for analysis.
- `Result/` - Analysis outputs, graphs, and final summary.
- `README.md` - Project documentation.
